### 🔬 Overview Técnica
Essa versão 0.4 foi desenvolvida com foco em **interoperabilidade de dados** e arquitetura de sistemas corporativos. Unindo o processamento científico em **Python** (via PyScript) com uma interface moderna, o sistema demonstra maturidade no tratamento de tipos e persistência, possui a geração de um script em SQL para transfêrencia.

---

### 🚀 Diferenciais da versão 0.3.2 (Oracle/SQL)
- **📟 SQL Script Engine**: Funcionalidade integrada para exportar o estado atual do `localStorage` para scripts de migração SQL (`INSERT INTO`), compatíveis com Oracle e SQL Server.
- **🛡️ Data Integrity**: Lógica de cálculo ($IMC = \frac{peso}{altura^2}$) isolada em engine Python, garantindo precisão numérica e tratamento de exceções.
- **🗃️ Relational-Ready Schema**: Estrutura de dados baseada em tipos primitivos compatíveis com bancos relacionais de larga escala.
